# MycountryGuide

> AI-powered market intelligence platform for business professionals

**Live:** [mycountryguide.de](https://mycountryguide.de)

---

## Overview

MycountryGuide is a comprehensive market intelligence platform for business professionals, investors, and consultants researching international markets. Aggregates live economic data from the World Bank alongside AI-generated country analysis — fully bilingual (EN/DE).

## Key Features

- **Interactive World Map** — country selection and navigation
- **AI Country Analysis** — powered by Anthropic Claude & Google Gemini
- **Economic Data Charts** — GDP, trade, growth indicators (World Bank API)
- **Country Comparison Tool** — side-by-side market analysis
- **Global Rankings** — country benchmarking system
- **Live Data Feeds** — exchange rates, weather, demographic data
- **Bilingual** — full English / German (DE/EN) language switcher
- **Dark Mode** — full dark/light theme support

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Framework | Next.js 16, TypeScript |
| AI | Anthropic Claude API, Google Generative AI |
| Data | World Bank API, REST Countries API |
| Mapping | Mapbox GL |
| Charts | Chart.js, Framer Motion |
| Search | Fuse.js |
| Styling | Tailwind CSS |
| Deployment | Vercel |
